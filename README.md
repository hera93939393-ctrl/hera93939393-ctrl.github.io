# hera93939393-ctrl.github.io

"기안 대신 '커밋'" — 현정의 AI 에이전트 공부 기록 블로그. GitHub Pages(Jekyll, minima 테마)로 운영합니다.

- 블로그 주소: https://hera93939393-ctrl.github.io
- 새 글 작성: `_posts/YYYY-MM-DD-제목.md` 형식으로 파일 추가 후 push하면 자동 반영됩니다.
- 글 목록 제목 옆에 이모지를 넣으려면, 파일 맨 위 front matter에 `icon: "🚀"`처럼 한 줄 추가하면 됩니다 (안 넣으면 기본값 📝가 표시됩니다).
- 글에 대표 이미지를 넣으려면 front matter에 `image: "이미지주소"`를 추가하세요. 글 목록에는 작은 썸네일로, 글 본문 상단에는 큰 배너로 자동으로 표시됩니다. (이미지는 GitHub 편집 화면에 드래그해서 올리면 생기는 주소를 그대로 쓰면 됩니다 — **단, 절대 `title:` 줄 안에는 넣지 마세요**, 문법이 깨집니다.)
- 목록 썸네일과 본문 배너를 서로 다른 이미지로 하고 싶으면, `thumb: "이미지주소"`를 추가로 넣으세요. `thumb`이 있으면 목록에는 `thumb`이, 본문 배너에는 `image`가 각각 따로 쓰입니다. (`thumb`을 안 넣으면 지금처럼 `image` 하나가 두 곳 다 씁니다.)
- 본문 배너 이미지는 기본적으로 1/3 크기로 작게 나오는데, 특정 글만 원래 크기(가로 꽉 차게)로 키우고 싶으면 front matter에 `banner_large: true`를 추가하세요. (이 글에만 적용되고, 다른 글은 그대로 1/3 크기 유지됩니다.)
